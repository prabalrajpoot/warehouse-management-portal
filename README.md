Warehouse Management Portal
Project Overview

Warehouse Management Portal is a cloud-based inventory and warehouse management system developed using FastAPI, React.js, PostgreSQL, Neon, Render, and Vercel.

It enables warehouse managers to manage inventory, dispatch, inspection, inward/outward stock, manpower, and users through a role-based authentication system.

Features
1.JWT Authentication
2.Role Based Login
3.Dashboard
4.Inventory Inward
5.Inventory Outward
6.Dispatch Management
7.Dispatch Return
8.Inspection
9.Sample Inspection
10.Warehouse Management
11.User Management
12.Activity Logs
13.Excel Upload
14.Responsive UI
